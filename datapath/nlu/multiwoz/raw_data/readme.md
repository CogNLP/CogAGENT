# The MultiWOZ Dataset
Download  MultiWOZ dataset .    
Put the raw data in path`datapath/nlu/multiwoz/raw_data`.   
Like the following form:

```angular2html
datapath
├─ nlu
│  ├─ multiwoz
│  │  ├─ raw_data
│  │  │  ├─ train.tsv
│  │  │  ├─ dev.tsv
│  │  │  ├─ test.tsv
│  │  │  └─ original
│  │  │  
```
